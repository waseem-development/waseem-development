## &nbsp;&nbsp;About

Co-founder of **ShalStack** — a product studio building AI-first software for real users, not investors.

I work at the overlap of **large language models**, **full-stack engineering**, and **Shariah-compliant financial technology**. Almost nobody is building seriously in that intersection. That's precisely why it matters.

No tutorial loops. No demo projects. If it doesn't run in production, it doesn't count.
